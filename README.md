# Flashcard App

This is the skeleton structure for a Java Swing flashcard application.

## Project layout

- `pom.xml` - Maven build descriptor
- `src/main/java` - Java source code
- `src/main/resources` - application resources and SQL files

## Modules

- `main` - application entry point
- `ui` - Swing user interface frames
- `model` - data model classes
- `dao` - data access objects
- `service` - business logic services
- `algorithm` - algorithm implementation for spaced repetition
